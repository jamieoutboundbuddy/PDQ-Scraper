# PDQ Scraper

A powerful web scraping tool for PDQ data collection and analysis.

## Features

- Fast and efficient web scraping
- Data extraction and transformation
- Error handling and retry logic
- Configurable settings

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/jamieoutboundbuddy/PDQ-Scraper.git
cd PDQ-Scraper
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```

## Configuration

Edit `config.yml` to customize scraper settings.
